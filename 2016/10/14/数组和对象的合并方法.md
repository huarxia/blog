数组合并：
```
var arr1 = [1];
var arr2 = [2];
var arr = arr1.concat(arr2);
console.log(arr);
```
对象合并：
```
var obj = { a: 1 };
var obj1 = {
  a:2,
  c:{
    d:1,
    e:{
      ff:1
    }
  }
};
var copy = Object.assign(obj1, obj);
console.log(copy)
```
