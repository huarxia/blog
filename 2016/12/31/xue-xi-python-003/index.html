動動手

  1. 001的动动手的题目吗？这一次要求使用变量，计算一年有多少秒？

我們來使用變量：

# @file:      變量計算一年多少秒
# @author:    花夏(liubiao@itosx.com)
# @version:   V0.0.1
# @date:      2016-12-31 16:28:36

daysPerYear = 365  
hoursPerDay = 24  
minutesPerHour = 60  
secondsPerMinutes = 60  
result = daysPerYear * hoursPerDay * minutesPerHour * secondsPerMinutes  
result = str(result)  
print('一年有：' + result + '秒')  
# 輸出：一年有：31536000秒
